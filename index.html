D’Angelo『Voodoo』をどう聴くか

Music / R&B / Soul / 2000s Listening

『Voodoo』は、過去のソウル・ミュージックを再現したアルバムではない。過去の身体性を、ヒップホップ以後の時間感覚の中で再配置した作品である。

この作品を聴くとき、まず耳に入るのは演奏のゆるさである。だが、そのゆるさは単なるラフさではない。ドラムは正確さからわずかに外れ、ベースは拍の中心を避けるように動く。歌は前に出すぎず、楽器の一部として沈み込む。

ここで重要なのは、70年代ソウルへの敬意ではなく、その敬意が一度ヒップホップの耳を通過していることだ。生演奏でありながら、ループのように聴こえる。人間が演奏しているのに、サンプルのように反復する。

『Voodoo』は、過去へ戻る作品ではない。過去を素材として、2000年のR&Bがどこまで遅く、重く、曖昧になれるかを試した作品である。

Prince以後のR&Bについて

Music / Prince / Funk / R&B

Princeの音楽は、ジャンルの融合というより、ジャンルが一人の身体に集まりすぎた状態として聴こえる。

ファンク、ロック、ポップ、ゴスペル、ジャズ、R&B。それらはPrinceの作品の中で整理されるのではなく、しばしば過剰なまま同居する。だからPrince以後のR&Bを考えることは、単に影響源をたどることではない。

重要なのは、R&Bが歌唱のジャンルであると同時に、身体、信仰、性、録音技術、ステージングを含む総合的な表現形式になっていくことだ。

『Lovesexy』と『The Rainbow Children』を同じ線で結ぶ必要はない。むしろ、それらは同じ人物の中で別々の時点に現れた、似ているが同一ではない問いとして置かれるべきである。

影響関係ではなく、配置として聴く

Essay / Archive / Relation / Map

作品は単独で存在しているように見える。しかし実際には、他の作品、時代、メディア、聴き手の記憶の中に置かれている。

あるアルバムが別のアルバムに影響を与えた、と言うことは簡単である。だが、その線はしばしば強すぎる。影響という言葉は、作品同士の関係を原因と結果に押し込めてしまう。

だから、ここでは作品を「配置」として扱う。ある作品がどの時代に現れ、どの作品と近く、どの作品とは遠いのか。どの線は明確で、どの線はまだ仮説にすぎないのか。

NOISEは、作品を評価する場所というより、作品の位置を考える場所である。

ポップはパッケージとして届く

Pop / Design / CD Jacket / Media Object

ポップ・ミュージックは音だけで届くわけではない。ジャケット、フォント、色、写真、ステッカー、開封体験まで含めて、ひとつの作品として受け取られる。

配信時代になっても、パッケージの感覚は消えていない。むしろ、物理的なCDやレコードの役割が弱まったぶん、画面の中でパッケージ性をどう作るかが重要になっている。

黒い帯、過剰なロゴ、細かいクレジット、バーコードのような記号は、情報であると同時に質感でもある。それらは作品を説明するのではなく、作品に触れたという感覚を作る。

ポップは曲として聴かれる前に、まずパッケージとして見られる。NOISEの画面もまた、記事の容器であると同時に、読む前の気分を作るジャケットである。

映画における視線と編集

Film / Gaze / Editing / Frame

映画の中で、誰が誰を見るのか。その視線の順番は、物語の説明以上に強く、観客の感情を組み立てている。

カメラが顔を映す。次に、その人物が見ているものを映す。さらに、その対象が見返す。こうした単純な切り返しの中で、観客は関係性を理解する。言葉より先に、視線が人物の距離を決める。

編集とは、出来事を短くする作業ではない。視線の向き、反応の間、沈黙の長さを並べ替え、観客が何を信じるかを決める作業である。

だから映画を見ることは、誰が画面の中心にいるかを見ることだけではない。誰が見られ、誰が見返し、誰の視線だけが切り落とされているのかを読むことでもある。

インターネットの記憶はなぜ淡いのか

Internet / Memory / Archive / Feed

インターネット上の記憶は、はっきり保存されているようで、実際にはいつも少しだけ薄い。見たことは覚えているのに、どこで見たのかを思い出せない。

フィードは出来事を時系列に並べるが、体験としては時系列を壊す。昨日の投稿、数年前の画像、誰かの引用、広告、保存したまま見返していないリンク。それらが同じ平面に置かれる。

その結果、記憶は情報ではなく質感として残る。特定の文章よりも、色味、画質、スクリーンショットの余白、サムネイルの並び方の方が長く残る。

インターネットの記憶が淡いのは、忘れているからではない。むしろ、多すぎる断片が重なりすぎて、ひとつの像として輪郭を失っているからである。

個人編集のための方法

Editorial / Personal Magazine / Method

個人で雑誌を作ることは、巨大なメディアを小さく模倣することではない。むしろ、自分の関心の速度に合わせて、記事、配置、デザインを同時に動かすことである。

編集とは、文章を並べることだけではない。何を一番上に置くのか。どの言葉を大きくするのか。どの記事の隣にどの図版を置くのか。そうした判断の積み重ねが、媒体の人格を作る。

個人編集の強さは、規模の小ささにある。すべてを説明しなくていい。毎回同じ読者を想定しなくていい。未完成な関心を、そのまま号として出すことができる。

NOISEは、完成されたメディアではなく、編集の実験場である。記事を書くこと、配置すること、画面を作ること。それらを分けずに扱うための場所である。
